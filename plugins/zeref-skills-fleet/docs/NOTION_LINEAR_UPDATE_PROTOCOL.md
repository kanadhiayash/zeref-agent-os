# Notion and Linear Update Protocol

Command Center: https://copper-tv-288.notion.site/Zeref-Skills-Fleet-Command-Center-358d695d836a81af9f6adf30770217c3

When connector access exists and the user has approved updates, update the relevant Notion/Linear records directly.

When connector access is unavailable, every skill must produce copy-ready blocks for:

- Notion update
- Linear issue/ticket
- Handoff summary

Do not create, move, delete, publish, or schedule without explicit approval.
